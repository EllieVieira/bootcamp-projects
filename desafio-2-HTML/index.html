Tags HTML

(Voltar)

HTML é a linguagem básica que usamos para criar páginas da web. Ela é formada por tags, que funcionam como blocos de construção. Cada tag tem um papel específico, como definir títulos, parágrafos, listas ou links. Algumas são de abertura e fechamento, enquanto outras são auto-fechadas. Usando essas tags da forma certa, consigo estruturar o conteúdo da minha página de maneira clara e organizada — e até deixá-lo mais bonito e interessante de ler.

"HTML é a espinha dorsal de todas as páginas da web."

O que são tags?

(Voltar)

Tags são como comandos que a gente usa no HTML para dizer ao navegador o que cada parte do conteúdo significa ou como ela deve aparecer na página. É como se cada tag fosse uma caixinha que envolve o conteúdo e diz: "ei, isso aqui é um título", ou "isso aqui é um parágrafo", ou ainda "isso é um link, clique aqui!".

Então, basicamente, as tags são a forma que o HTML usa para estruturar e organizar tudo o que aparece em uma página.

Tags de formatação

(Voltar)

As tags de formatação são aquelas que usamos para mudar a aparência do texto dentro do HTML. Elas não mudam a estrutura da página, mas servem para destacar partes específicas, dar ênfase ou deixar a leitura mais visual e organizada.

Por exemplo, se eu quiser deixar algo em negrito, uso a tag <strong>, e para deixar em itálico, uso a <i>. Tem também a <u>, que sublinha, e a <mark>, que marca o texto como se fosse um marca-texto.

Elas são úteis quando eu quero chamar atenção para uma palavra importante, fazer uma observação discreta com a tag <small>, ou mostrar um trecho de código com <code>. A ideia é que o conteúdo continue sendo o mesmo, mas com um toque visual que faz diferença na experiência de quem lê.

Tag <Font>

Essa tag é considerada obsoleta no HTML5. O ideal é usar CSS para alterar fontes, tamanhos e cores, mas em projetos simples, ainda pode ser usada com cuidado.

Tag <Del>

Quando quero mostrar algo que foi "riscado", como se estivesse corrigindo um erro ou fazendo uma edição visível. Fica ótimo em explicações ou exemplos.

Tag <Abbr>

Quando uso uma sigla ou abreviação no texto, posso colocar o significado completo com a tag <abbr>. Assim, se alguém passar o mouse, aparece a explicação.

Observações
(Voltar)

Aprender e praticar o uso das tags HTML nos ajuda a entender melhor como uma página da web funciona por trás dos bastidores. As tags são simples, mas poderosas, e com o tempo a gente vai pegando o jeito de usar cada uma com mais naturalidade.

Uma dica importante: sempre que eu quiser mostrar o código como texto, preciso lembrar de usar as entidades HTML, como "&lt;" para < e "&gt;" para >. Isso evita que o navegador interprete a tag e ajuda quem está lendo a visualizar o código direitinho.

HTML é uma base essencial, e quanto mais se pratica, mais confiante ficamos na criação das nossas próprias páginas.
